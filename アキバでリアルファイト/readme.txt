// 操作方法
//====================================================================================
[コントローラー]
・移動　左ステック
・視点　右ステック
・通常攻撃　Ａボタン
・敵投げ　Yボタンでつかみ、掴んだ状態でYボタンで投げる　スティックをまわすと少し投げる距離が伸びる
・回避　Xボタンでプレイヤーの向いている方向に回避
・ものを持って投げる　LBボタンでつかんで、掴んだ状態でLBボタンでその場に叩きつける　Aボタンで掴んだもので攻撃
・ヒートアクション　もの持っている状態で敵に近づいてYボタン


[キーボード、マウス]
・移動　ＷＡＳＤ
・視点　マウス
・通常攻撃　右クリック
・敵投げ　Ｅキーでつかみ、掴んだ状態でＥキーで投げる AキーorDキーで回す方向を決める(おしっぱで回る)
・回避　LSHIFTでプレイヤーの向いている方向に回避
・ものを持って投げる　Fキーでつかんで、掴んだ状態でFキーでその場にたたきつける　右クリックで掴んだもので攻撃
・ヒートアクション　もの持っている状態で敵に近づいてEキー
